# CSV Sales Analyzer

Exploratory data analysis tool for sales data built with Python.
Processes CSV files to generate metrics, visualizations, and automated reports.

## Tech Stack

- Python 3.x
- pandas
- matplotlib
- seaborn

## Project Structure

proyecto-ventas/

├── data/           # Input data in CSV format
├── notebooks/      # Exploratory analysis in Jupyter
├── outputs/        # Generated charts and reports
├── requirements.txt
└── README.md

## Installation

1. Clone the repository:
```bash
   git clone git@github.com:edusnow02/proyecto-ventas.git
```

2. Create virtual environment:
```bash
   python -m venv .venv
   source .venv/bin/activate
```

3. Install dependencies:
```bash
   pip install -r requirements.txt
```

## Usage

Open the main notebook in Jupyter:

```bash
jupyter notebook notebooks/
```
